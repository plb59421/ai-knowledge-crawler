# Knowledge Base Runtime Directory

[中文](#中文) | [English](#english)

---

## 中文

`knowledge_base/` 是爬虫运行时使用的本地文件数据库。这里的数据由爬虫、AI 分析、标签评分和报告流程生成。

默认不提交运行时数据。

### 子目录

- `by_source/`: 按来源和日期保存的文章 JSON，是主存储。
- `by_time/`: 按时间组织的派生视图。
- `by_topic/`: 按主题组织的派生视图。
- `index/`: URL、去重和搜索索引。
- `metadata/`: 抓取历史和日报。
- `exports/`: 静态报告数据、Markdown 导出和前端构建产物。

需要提交稳定样例时，请放入 `tests/fixtures/`。

---

## English

`knowledge_base/` is the local file database used at runtime. Data here is generated by crawling, AI analysis, tagging, ranking, and reporting flows.

Runtime data is ignored by default.

### Subdirectories

- `by_source/`: Canonical article JSON by source and date.
- `by_time/`: Time-based derived view.
- `by_topic/`: Topic-based derived view.
- `index/`: URL, deduplication, and search indexes.
- `metadata/`: Crawl history and daily reports.
- `exports/`: Static report data, Markdown exports, and frontend build artifacts.

Committed sample data should live in `tests/fixtures/`.
