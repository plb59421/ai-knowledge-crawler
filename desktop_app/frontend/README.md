# Desktop Frontend Adapter

[中文](#中文) | [English](#english)

---

## 中文

本目录用于存放桌面应用专用的前端适配层。

当前主报告前端仍位于 `web/report/`。桌面版 v1 可以复用主前端构建产物；如果后续需要桌面专用设置页、任务页或日志页，应在本目录中隔离实现。

---

## English

This directory stores the desktop-specific frontend adapter layer.

The main report frontend still lives in `web/report/`. Desktop v1 may reuse the main frontend build output. If desktop-specific settings, tasks, or logs pages are needed later, they should be isolated here.
