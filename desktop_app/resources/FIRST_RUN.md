# First Run Guide

[中文](#中文) | [English](#english)

---

## 中文

桌面应用首次启动时应完成：

1. 创建系统用户数据目录。
2. 初始化本地配置文件。
3. 提示用户配置 API Key。
4. 检查代理和浏览器运行时依赖。
5. 启动本地 API 并打开报告页面。

---

## English

On first launch, the desktop app should:

1. Create the system user data directory.
2. Initialize the local configuration file.
3. Prompt the user to configure an API key.
4. Check proxy and browser runtime dependencies.
5. Start the local API and open the report page.
